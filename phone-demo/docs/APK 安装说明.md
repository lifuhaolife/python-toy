# APK 安装说明

> 儿童智能语音玩具 - 快速安装指南

---

## 📱 方式一：USB 传输安装 (推荐)

### 步骤 1：连接手机

1. 使用 USB 数据线连接手机和电脑
2. 手机选择"文件传输"模式

### 步骤 2：复制 APK

将以下文件复制到手机：
```
phone-demo/app/bin/toyphone-1.0.0-debug.apk
```

### 步骤 3：手机安装

1. 打开手机文件管理器
2. 找到 `toyphone-1.0.0-debug.apk`
3. 点击安装
4. 等待安装完成

---

## 📱 方式二：HTTP 下载安装

### 步骤 1：电脑启动 HTTP 服务器

```bash
cd phone-demo/app/bin
python -m http.server 8080
```

### 步骤 2：手机浏览器访问

```
http://电脑 IP:8080/toyphone-1.0.0-debug.apk
```

**查看电脑 IP**:
```bash
# Windows
ipconfig

# Linux/Mac
ifconfig
```

### 步骤 3：下载安装

1. 点击下载链接
2. 下载完成后点击安装
3. 如果提示"未知来源"，点击"设置"
4. 开启"允许从此来源安装"
5. 返回继续安装

---

## 📱 方式三：ADB 安装 (开发者)

### 前提条件

- 手机开启 USB 调试
- 电脑安装 ADB 工具

### 安装命令

```bash
cd phone-demo/app
adb install bin/toyphone-1.0.0-debug.apk
```

### 如果安装失败

```bash
# 允许覆盖安装
adb install -r bin/toyphone-1.0.0-debug.apk

# 允许降级安装
adb install -d bin/toyphone-1.0.0-debug.apk
```

## 🔧 常见问题

### Q1: 无法安装 APK

**解决方法**:
```
设置 → 安全 → 未知来源 → 允许
或
设置 → 应用 → 特殊应用权限 → 安装未知应用 → 允许浏览器/文件管理器
```

### Q2: 签名错误

**原因**: 之前安装过不同签名的版本

**解决**:
```
1. 卸载旧版本
2. 重新安装新版本
```

### Q3: 解析包错误

**原因**: Android 版本过低

**解决**:
- 检查手机 Android 版本 ≥ 5.0
- 重新下载完整 APK

### Q4: 应用闪退

**解决**:
```
1. 清除应用数据
   设置 → 应用 → 儿童智能语音玩具 → 存储 → 清除数据

2. 检查服务器是否启动

3. 查看日志:
   adb logcat | grep toyphone
```

## 📲 首次使用

### 1. 打开应用

在手机桌面找到图标并打开

### 2. 授予权限

- 麦克风权限 → 允许
- 网络权限 → 允许

### 3. 连接服务器

1. 点击"连接服务器"
2. 状态变为"已连接"(绿色)

### 4. 开始对话

1. 按住"🎤 按住说话"
2. 对着手机说话
3. 松开按钮
4. 等待 AI 回复
