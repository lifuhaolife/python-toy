# Pydroid 3 快速安装指南

> 无需编译！5 分钟在旧手机上运行 APP

---

## 📱 方案说明

**Pydroid 3** 是一个运行在 Android 手机上的 Python IDE，可以直接运行 Python 代码，**无需编译 APK**。

### 优势

| 优势 | 说明 |
|------|------|
| ✅ 无需编译 | 不需要 Buildozer、SDK、NDK |
| ✅ 5 分钟完成 | 安装→复制→运行 |
| ✅ 可调试 | 直接查看错误日志 |
| ✅ 可修改 | 随时修改代码测试 |
| ✅ 零门槛 | 不需要技术背景 |

### 劣势

| 劣势 | 说明 |
|------|------|
| ❌ 需要 Pydroid 3 | 需安装额外 APP |
| ❌ 依赖网络 | 首次安装依赖需联网 |
| ❌ 非独立 APP | 在 Pydroid 3 内运行 |

---

## 📥 步骤 1：安装 Pydroid 3

### 方式 A：Google Play (推荐)

1. 打开手机上的 **Google Play 商店**
2. 搜索 **"Pydroid 3"**
3. 点击 **安装**
4. 等待安装完成

### 方式 B：官网下载

1. 浏览器访问：https://pydroid.net/
2. 下载 APK 文件
3. 安装到手机

### 方式 C：第三方市场

在以下市场搜索 "Pydroid 3"：
- 豌豆荚
- 应用宝
- 华为应用市场
- 小米应用商店

---

## 📁 步骤 2：复制代码到手机

### 方式 A：USB 传输

1. **USB 连接手机和电脑**
2. **手机选择"文件传输"模式**
3. **复制文件**：
   ```
   从电脑复制：
   C:\Users\lenovo\Claudes\toy\phone-demo\app\
   
   到手机：
   手机存储/Download/toyphone/
   ```

4. **确认文件结构**：
   ```
   手机存储/Download/toyphone/
   ├── main.py
   ├── toyphone.kv
   ├── requirements.txt
   └── ... (其他文件)
   ```

### 方式 B：HTTP 传输

**电脑上执行**：
```bash
cd C:\Users\lenovo\Claudes\toy\phone-demo\app
python -m http.server 8080
```

**手机浏览器访问**：
```
http://电脑 IP:8080/
```

长按文件下载，保存到：
```
手机存储/Download/toyphone/
```

### 方式 C：微信/QQ 传输

1. 电脑上压缩 `app` 文件夹为 `toyphone.zip`
2. 通过微信/QQ 发送到手机
3. 手机解压到 `Download/toyphone/`

---

## 📦 步骤 3：安装 Python 依赖

### 打开 Pydroid 3

1. 在手机桌面找到 **Pydroid 3** 图标
2. 点击打开

### 安装依赖

1. 点击左上角 **菜单按钮** (☰)
2. 选择 **Pip**
3. 在输入框中输入：
   ```
   kivy aiohttp pyaudio
   ```
4. 点击 **Install** 按钮
5. 等待安装完成（约 2-5 分钟）

### 安装成功标志

看到以下信息表示成功：
```
Successfully installed kivy-2.x.x aiohttp-3.x.x pyaudio-0.2.x
```

---

## ▶ 步骤 4：运行 APP

### 打开文件

1. 在 Pydroid 3 中，点击 **文件图标** (📁)
2. 浏览到：`手机存储/Download/toyphone/`
3. 点击 **main.py** 文件
4. 文件内容会显示在编辑器中

### 运行

1. 点击右下角 **▶ 运行按钮**
2. 等待几秒钟
3. APP 界面会出现

### 首次运行权限

如果是首次运行，Pydroid 3 会请求权限：
- **麦克风权限** → 允许 ✅
- **存储权限** → 允许 ✅

---

## 🔧 步骤 5：配置和测试

### 修改服务器地址

在 Pydroid 3 中编辑 `main.py`：

```python
# 找到第 26 行左右
SERVER_HOST = "192.168.1.50"  # 改为你的电脑 IP
```

**查看电脑 IP**：
```bash
# Windows
ipconfig
# 找到 IPv4 地址，如 192.168.1.100
```

### 启动服务器

**电脑上**：
```bash
cd phone-demo/server
start.bat
```

### 测试连接

1. 在 APP 中点击 **"连接服务器"**
2. 状态变为 **"已连接"** (绿色)
3. 按住 **"🎤 按住说话"** 测试对话

## 🐛 常见问题

### Q1: Pip 安装依赖失败

**解决方法**：
```
1. 检查网络连接
2. 重试安装
3. 如果还失败，尝试单独安装：
   - 先安装 kivy
   - 再安装 aiohttp
   - 最后安装 pyaudio
```

### Q2: 运行时提示找不到模块

**解决方法**：
```
1. 确认依赖已安装成功
2. 重启 Pydroid 3
3. 重新打开 main.py
4. 再次运行
```

### Q3: 无法连接服务器

**检查**：
```
1. 服务器是否运行
2. 手机和电脑是否同一 WiFi
3. IP 地址是否正确
4. 防火墙是否关闭
```

### Q4: 麦克风无法使用

**解决方法**：
```
设置 → 应用管理 → Pydroid 3 → 权限 → 麦克风 → 允许
```
