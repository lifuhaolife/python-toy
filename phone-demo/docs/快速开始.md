## 方案 A：电脑测试 (5 分钟)

### 步骤 1：启动服务器 (2 分钟)

```bash
# 1. 打开命令行
cd C:\Users\lenovo\Claudes\toy\phone-demo\server

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置 API 密钥
# 复制 .env.example 为 .env
# 编辑 .env 填入 AI_API_KEY

# 4. 启动服务器
python server.py
```

### 步骤 2：运行 APP (3 分钟)

```bash
# 1. 打开新命令行
cd C:\Users\lenovo\Claudes\toy\phone-demo\app

# 2. 安装依赖
pip install kivy aiohttp pyaudio

# 3. 运行 APP
python main.py
```

### 步骤 3：测试对话

1. 点击"连接服务器"
2. 按住"🎤 按住说话"
3. 说话后松开
4. 等待 AI 回复

## 🔑 获取 AI API 密钥

### 智谱 AI (推荐)

1. 访问 https://open.bigmodel.cn/
2. 注册/登录账号
3. 进入控制台
4. 创建 API Key
5. 复制到 `.env` 文件

```ini
AI_API_KEY=your_api_key_here
```

### 免费替代：使用模拟响应

临时测试可以不配置 API Key，服务器会返回模拟响应：

```python
# 服务器会自动降级为模拟模式
user_text = "你好呀，小朋友！"
ai_response = "你好！今天想听什么故事呢？"
```

## 🐛 快速故障排查

### 无法连接服务器

```
1. 检查服务器是否运行
2. 检查 IP 地址是否正确
3. 关闭防火墙测试
4. 确保同一 WiFi 网络
```

### 录音无声

```
1. 检查麦克风权限
2. 检查系统音量
3. 重启 APP 测试
```

### TTS 无声

```
1. 检查手机音量
2. 检查网络连接
3. 查看服务器日志
```
