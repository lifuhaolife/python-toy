# 故障排查指南

> 儿童智能语音玩具 - 常见问题诊断

---

## 🔍 检查清单

### 1. 服务器状态检查

**检查服务器是否运行**：
```bash
curl http://localhost:8000/health
```

**预期响应**：
```json
{"status":"healthy","connections":0}
```

**如果失败**：
```bash
# 启动服务器
cd phone-demo/server
python server.py
```

---

### 2. APP 连接问题

#### 症状：点击"连接服务器"后显示"连接失败"

**可能原因 1：服务器未启动**
```
解决：确保服务器正在运行
```

**可能原因 2：服务器地址错误**
```
检查：app/main.py 第 54 行
SERVER_HOST = "localhost"  # 电脑测试
# 或
SERVER_HOST = "192.168.1.XXX"  # 手机测试
```

**可能原因 3：防火墙阻止**
```
解决：关闭防火墙或添加例外
Windows 防火墙 → 允许应用通过防火墙 → Python
```

---

### 3. 中文乱码问题

#### 症状：UI 显示方框或问号

**原因**：Kivy 默认字体不支持中文

**解决**：代码已自动注册中文字体

如果仍然乱码，检查：
```python
# app/main.py 开头部分
LabelBase.register(name='Chinese', fn_regular='...')
```

---

### 4. 日志查看

#### APP 日志位置

**UI 日志**：APP 界面底部日志区域

**控制台日志**：启动 APP 的命令行窗口

**关键日志格式**：
```
[APP LOG] [HH:MM:SS] 消息内容
[APP ERROR] 错误详情
```

#### 服务器日志

```
INFO:     Started server process [PID]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

---

### 5. 常见错误代码

#### ConnectionRefusedError

```
错误：ConnectionRefusedError: Cannot connect to host
原因：服务器未运行或地址错误
解决：检查服务器状态和地址配置
```

#### WebSocket 连接失败

```
错误：WebSocket connection failed
原因：WebSocket 端点不可用
解决：检查服务器日志，确认 WebSocket 路由正确
```

## 🛠️ 诊断命令

### 检查端口
```bash
netstat -ano | findstr :8000
```

### 检查进程
```bash
tasklist | findstr python
```

### 测试 WebSocket
```bash
# 使用 Python 测试
python -c "import asyncio; import websockets; asyncio.run(websockets.connect('ws://localhost:8000/ws/audio/test'))"
```
