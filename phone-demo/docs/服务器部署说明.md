# 服务器部署说明

> 完整部署流程 - 从环境安装到运行

---

## 📦 环境准备

### 1. Python 环境

```bash
# 检查 Python 版本（需要 3.9+）
python --version

# 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate  # Windows
```

### 2. 安装依赖

```bash
cd server
pip install -r requirements.txt
```

**依赖包括**:
- fastapi, uvicorn - Web 框架
- edge-tts - TTS 语音合成
- faster-whisper - Whisper 语音识别
- httpx - HTTP 客户端

## 🚀 启动服务器

### 配置环境变量

创建 `.env` 文件：
```ini
# AI 服务配置（必需）
AI_API_KEY=your_api_key_here

# 服务器配置
HOST=0.0.0.0
PORT=8000
DEBUG=true
```

**获取 API Key**:
1. 访问 https://open.bigmodel.cn/
2. 注册账号
3. 创建 API Key
4. 填入 `.env` 文件

### 启动命令

cd server

## 🐛 常见问题

### Q1: 模型下载失败

**错误**: `ConnectionError`, `TimeoutError`

**解决**:
1. 使用镜像下载：`set HF_ENDPOINT=https://hf-mirror.com`
2. 运行下载脚本：`python download_whisper_model.py`
3. 检查网络连接

---

### Q2: 模型加载失败

**错误**: `Model not found`

**解决**:
```bash
# 检查模型目录
dir models\whisper-tiny

# 应该包含以下文件:
# - config.json
# - model.bin
# - tokenizer.json
# - vocabulary.txt
```

---

### Q3: API Key 无效

**错误**: `401 Unauthorized`

**解决**:
1. 检查 `.env` 文件是否存在
2. 检查 `AI_API_KEY` 是否正确
3. 重启服务器

---

### Q4: 端口被占用

**错误**: `Address already in use`

**解决**:
```bash
# 查看占用端口的进程
netstat -ano | findstr :8000

# 杀死进程
taskkill /F /PID <进程 ID>

# 或修改端口
# 编辑 .env 文件，修改 PORT=8001
```

## 🎯 生产环境部署

### 使用 Gunicorn

```bash
pip install gunicorn

gunicorn server:app \
  --bind 0.0.0.0:8000 \
  --workers 4 \
  --worker-class uvicorn.workers.UvicornWorker
```

### 使用 Docker

```dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["python", "server.py"]
```
