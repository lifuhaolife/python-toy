## 📦 环境要求

- Python 3.9+
- faster-whisper 已安装 (`pip install -r requirements.txt`)

## 🎯 模型下载（必需）

### 方式 1: 使用镜像下载（推荐）

```bash
cd server

# 设置镜像地址
set HF_ENDPOINT=https://hf-mirror.com

# 运行下载脚本
python download_whisper_model.py

# 输入 y 确认下载
```

**下载完成日志**:
```

模型路径：C:\...\server\models\whisper-tiny
```

## 🐛 故障排查

### Q1: 模型加载失败

**错误**: `模型不存在`

**解决**:
```bash
# 检查模型目录
dir server\models\whisper-tiny

# 如果目录不存在或为空，重新下载
cd server
set HF_ENDPOINT=https://hf-mirror.com
python download_whisper_model.py
```

---

### Q2: 识别结果为空

**日志**: `[ASR] 未识别到有效语音（返回空字符串）`

**原因**:
1. 录音太短（< 1 秒）
2. 说话声音太小
3. 环境噪音太大

**解决**:
1. 说话时间稍长（2-3 秒）
2. 对着麦克风说话
3. 在安静环境测试

---

### Q3: 识别不准确

**可能原因**:
1. 模型太小（tiny 模型准确率约 80%）
2. 口音较重
3. 语速太快

**解决**:
1. 说话清晰、语速适中
2. 在安静环境测试
3. 更换更大模型（可选）

## 📝 快速部署流程

```bash
# 1. 进入服务器目录
cd server

# 2. 安装依赖
pip install -r requirements.txt

# 3. 下载模型
set HF_ENDPOINT=https://hf-mirror.com
python download_whisper_model.py

# 4. 配置环境变量
copy ..\.env.example .env
# 编辑 .env 填入 AI_API_KEY

# 5. 启动服务器
python server.py

# 6. 验证
curl http://localhost:8000/health
```
